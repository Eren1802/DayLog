# DayLog

A daily productivity logger inspired by agile standup format.
Users log what they did, what's pending, and their mood — every day.

## Features
- Daily log entries with mood tracking
- Streak system to stay consistent
- Weekly stats dashboard
- One log per day rule
- User authentication

## Tech Stack
- Python + Django
- MongoDB (via Djongo)
- HTML + CSS

## Run Locally
pip install -r requirements.txt
python manage.py runserver
